Task1 No difference! In both cases, return confirm('Did parents allow you?') executes exactly when the if condition is falsy. Task2 Using a question mark operator '?': function checkAge(age) { return (age > 18) ? true : confirm('Did parents allow you?'); } Using OR || (the shortest variant): function checkAge(age) { return (age > 18) || confirm('Did parents allow you?'); } Note that the parentheses around age > 18 are not required here. They exist for better readability. Task3 A solution using if: function min(a, b) { if (a < b) { return a; } else { return b; } } A solution with a question mark operator '?': function min(a, b) { return a < b ? a : b; } P.S. In the case of an equality a == b it does not matter what to return. Task4
